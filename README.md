```
/
├── app/
│   ├── layout.tsx
│   └── page.tsx
├── components/
├── lib/
├── public/
└── package.json
```

## Commands

| Command        | Action                                       |
| :-------------- | :-------------------------------------------- |
| `pnpm install`  | Installs dependencies                        |
| `pnpm dev`      | Starts local dev server at `localhost:3000`  |
| `pnpm build`    | Build static site to `./out/`                |
| `pnpm start`    | Starts production server (not used for deploy) |
